A graph is a mathematical structure used to represent relationships between pairs of elements. A graph consists of a set of nodes and a set of links connecting the nodes.

In data science, graphs can be used to represent and analyze relationships between different data elements. For example, graphs can be used to represent friendship relationships in a social network, collaboration relationships in a set of documents, or dependency relationships in a set of tasks. Graphs are a very useful tool in data science and can be used for many different purposes, including data visualization, pattern identification, and communication of results.

In this case the application we have sought is to show the relationships between users and the businesses to which they have written a review.

# Create an empty graph
G_200 = nx.Graph()
for index, row in tqdm(df_network_200.iterrows()):
    # Add the nodes to the graph
    G_200.add_node(row['business_id'])
    G_200.add_node(row['user_id'])
    # Add the edge to the graph and assign the weight indicated in stars_x
    G_200.add_edge(row['business_id'], row['user_id'], weight=row['stars_x'])

G_nodes = G_200.number_of_nodes()
G_edges = G_200.number_of_edges()
print("Nodes = ", G_nodes, " Edges = ", G_edges)

To visualize the graph, we have executed the gexf file with the Gephi program.


We have used the 'Force Atlas' distribution algorithm, parameterizing it to obtain the most appropriate visualization in order to explain the graph properly.


The larger nodes in red represent the 3 selected users, the nodes in blue represent the evaluated businesses.


The edges that go from the nodes of the users to the nodes of the businesses represent the score that has been introduced in the review, the color of this edge indicates the value of the score, being red a negative score 1 and gradually modifying its color up to dark green that indicates a very positive score 5.


We can see that there are several reddish edges that converge in the same business, this allows us to conclude that if several reviews from different users agree on a negative score, it is very likely that the experience in that business is not satisfactory. In the same way, it is possible to locate businesses in which very positive user reviews converge, leading to the conclusion that the experience in that business is positive.
